# rangeScan

Tool for discover active hosts in an IP range.

# How to install

  [1] git clone https/github.com/Sxmpl3/rangeScan
  [2] cd rangeScan
  [3] chmod +x install.sh
  [4] bash install.sh
  
 # How to use
 
 [1] You need to have tool installed
 [2] rangeScan -i (IP address without host: 192.168.1)
